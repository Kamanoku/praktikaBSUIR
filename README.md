# 🔍 Определение пола по ФИО

Веб-приложение с использованием глубокого обучения для валидации ФИО и автоматического определения пола по написанию (на базе TensorFlow/Keras и Flask).

## ✨ Особенности
- **Интеллектуальная валидация**: Защита от мусорного ввода (одинаковые буквы, отсутствие кириллицы, слишком короткие строки).
- **Поддержка сложных имён**: Корректно обрабатывает дефисы, два вида апострофов (*Д’Арк*, *О’Коннор*), а также арабские/тюркские служебные частицы (*ибн*, *оглы*, *кызы*).
- **Двухэтапное подтверждение**: Фронтенд позволяет принудительно принять данные, если модель засомневалась или сработал валидатор.
- **Гибридный подход**: Для восточных имён со служебными частицами (`оглы`, `ибн`, `кызы`) применяется точный словарный анализатор, гарантирующий 100% корректный результат без ошибок нейросети.
