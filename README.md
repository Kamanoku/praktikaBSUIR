# 🔍 Определение пола по ФИО

Веб-приложение с использованием глубокого обучения для валидации ФИО и автоматического определения пола по написанию (на базе TensorFlow/Keras и Flask).

## ✨ Особенности
- **Интеллектуальная валидация**: Защита от мусорного ввода (одинаковые буквы, отсутствие кириллицы, слишком короткие строки).
- **Двухэтапное подтверждение**: Фронтенд позволяет принудительно принять данные, если модель засомневалась или сработал валидатор.

## 🚀 Быстрый старт

### 1. Подготовка окружения
Создайте виртуальное окружение и установите зависимости:
```bash
python -m venv .venv
source .venv/bin/activate  # для Linux/macOS
# или .venv\Scripts\activate  # для Windows

pip install flask flask-cors tensorflow numpy kaggle
```

# Скачивание датасета
```bash
kaggle datasets download -d rai220/russian-cyrillic-names-and-sex
```

# Распаковка архива (для Linux/macOS)
```bash
unzip russian-cyrillic-names-and-sex.zip
```
# или распакуйте архив russian-cyrillic-names-and-sex.zip вручную в корень проекта

# Обучение модели
- **Если вы хотите сгенерировать файлы весов и словарь символов заново, запустите скрипт обучения:**
```bash
python train.py
```

# Запуск сервера
- **Запустите Flask бэкенд:**
```bash
python app.py
```

# Запуск веб-окна
- **Просто откройте файл index.html в любом современном браузере. Компиляция или сборка фронтенда не требуется.**
